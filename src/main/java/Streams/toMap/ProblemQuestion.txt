EN:
Student ID Mapping:
Create a class called Student with attributes:
id (Integer)
name (String)
course (String)

Requirements:
Create a list with 10 students.
Using toMap():
Create a Map<Integer, String> mapping student ID to student name.
Create a Map<Integer, Student> mapping student ID to the complete Student object.
Create a list with some students having duplicate courses, then create a Map<String, String> mapping course to student name (use merge function to keep the first student when duplicates occur).
Print all maps.

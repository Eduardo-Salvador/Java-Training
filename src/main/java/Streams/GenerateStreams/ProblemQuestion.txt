EN:
Number Generation:

Requirements:

Using Stream.generate():
Generate 15 random integers between 1 and 100.
Filter only even numbers.
Collect into a list and print

Using Stream.iterate():
Generate the first 20 multiples of 3 (3, 6, 9, 12...).
Filter only numbers greater than 30.
Print the result.

Using IntStream.range():
Generate numbers from 1 to 50.
Filter only prime numbers.
Calculate the sum of all prime numbers.
Print the sum.

PT:
Geração de Números:

Requisitos:

Usando Stream.generate():
Gere 15 números inteiros aleatórios entre 1 e 100.
Filtre apenas números pares.
Colete em uma lista e imprima.

Usando Stream.iterate():
Gere os primeiros 20 múltiplos de 3 (3, 6, 9, 12...).
Filtre apenas números maiores que 30.
Imprima o resultado.

Usando IntStream.range():
Gere números de 1 a 50.
Filtre apenas números primos.
Calcule a soma de todos os números primos.
Imprima a soma.

EN-US:
Create an age calculator that:

Uses Calendar.getInstance() to get the current date
Creates another Calendar with set() for a birth date (e.g., 03/15/1990)
Uses get(Calendar.YEAR) and get(Calendar.MONTH) to calculate age
Uses add() to add 10 years to the birth date
Shows how many days until the next birthday using get(Calendar.DAY_OF_YEAR)

Comment: Remember that the month starts at 0! January = 0, December = 11. This is very confusing and was fixed in java.time.

PT-BR:
Crie um calculador de idade que:

Use Calendar.getInstance() para pegar a data atual
Crie outro Calendar com set() para uma data de nascimento (ex: 15/03/1990)
Use get(Calendar.YEAR) e get(Calendar.MONTH) para calcular a idade
Use add() para adicionar 10 anos à data de nascimento
Mostre quantos dias faltam para o próximo aniversário usando get(Calendar.DAY_OF_YEAR)

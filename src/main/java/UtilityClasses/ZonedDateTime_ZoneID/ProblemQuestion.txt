EN-US:
Create a time zone converter:

Create a ZonedDateTime for São Paulo: 12/25/2025 at 10:00 AM
Convert this same moment to:
    Tokyo (Japan)
    New York (USA)
    London (UK)
Show the hour difference between each zone
Use withZoneSameInstant() to convert

Comment: ZonedDateTime stores the complete zone (America/Sao_Paulo), while OffsetDateTime stores only the offset (-03:00). Use ZonedDateTime when you need correct time zone conversions!

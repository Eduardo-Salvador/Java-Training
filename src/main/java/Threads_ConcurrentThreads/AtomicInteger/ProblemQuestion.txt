Concurrent Counter with AtomicInteger

You are building a simple metrics system that counts how many users accessed a website.

Create a program where:
5 threads simulate users accessing the system.
Each thread increments a shared counter 1,000 times.
The counter must be thread-safe.
Use AtomicInteger from java.util.concurrent.atomic.
At the end, print the final counter value.

Requirements
Do NOT use synchronized.
Do NOT use locks.
Use AtomicInteger.
Wait for all threads to finish before printing the result.
The result must be deterministic.

Expected Output
If everything is correct:
Final counter: 5000

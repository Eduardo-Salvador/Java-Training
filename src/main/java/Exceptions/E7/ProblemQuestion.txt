Exceptions in business rules:
Create a BankAccount class that throws a custom  BalanceInsufficientException exception when attempting to withdraw an amount greater than the available balance, handling this exception in the main method.

Exceções nas regras de negócio:
Crie uma classe BankAccount que lance uma exceção personalizada BalanceInsufficientException ao tentar sacar um valor maior que o saldo disponível, tratando essa exceção no método principal.

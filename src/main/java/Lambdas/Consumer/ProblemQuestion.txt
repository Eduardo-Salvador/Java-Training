Product Processing System with Consumer:

Create a class called Product with the following attributes:
name (String)
price (Double)

Requirements:
In the main class, create a list of products with different names and prices.
Create a utility method called processProducts(List<Product> products, Consumer<Product> action) that receives:
The list of products.
A Consumer<Product> that defines the action to be applied to each product.
The method should iterate over the list and apply the Consumer to each item.
In the main class, use lambda expressions to create different Consumer<Product> and test the method:
One that prints the product name and price.
One that applies a 10% discount to the price and prints the new value.
One that prints only the names in uppercase letters.
Execute the processProducts method with each Consumer and observe the results.

Extra Challenge:
Create a list of Consumer<Product> and apply them all sequentially to each product (e.g., first print, then apply discount, then print in uppercase).
Combine Consumer using the andThen() method to chain actions into a single process.

Sistema de Processamento de Produtos com Consumer:

Crie uma classe chamada Product com os seguintes atributos:
nome (String)
preço (Double)

Requisitos:
Na classe principal, crie uma lista de produtos com nomes e preços diferentes.
Crie um método utilitário chamado processProducts (Lista<Product> produtos, Consumer<Product> ação) que receba:
A lista de produtos.
Um Consumer<Product> que define a ação a ser aplicada a cada produto.
O método deve iterar sobre a lista e aplicar o Consumer a cada item.
Na classe principal, use expressões lambda para criar diferentes Consumer<Product> e teste os métodos:
Um que imprime o nome e o preço do produto.
Um que aplica um desconto de 10% ao preço e imprime o novo valor.
Um que imprime apenas os nomes em letras maiúsculas.
Execute o método processProducts com cada Consumidor e observe os resultados.

Desafio Extra:
Crie uma lista de Consumidor<Produto> e aplique-os sequencialmente a cada produto (por exemplo, primeiro imprima, depois aplique o desconto e, em seguida, imprima em maiúsculas).
Combine os Consumer usando o método andThen() para encadear as ações em um único processo.lista com os nomes em maiúsculas de todos os produtos com preço acima de R$ 50,00.
Imprima a lista final.

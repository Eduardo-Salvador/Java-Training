Exercise: Phone Book with TreeMap (NavigableMap)
Create a program that manages a phone book using TreeMap or NavigableMap.

Requirements:
Create a Contact class with:
String name
String phoneNumber
String email

In the main program:
Create a NavigableMap<String, Contact>, where the key will be the contact's name and the value will be the Contact object itself.
Add at least 5 contacts.
Display all contacts in alphabetical order by name.
Show the first and last contact in the phone book.
Given a name, show:
the immediately preceding contact (lowerEntry)
the immediately following contact (higherEntry)
Display the phone book in reverse order (use descendingMap()).

Extra Challenges:
Allow removing a contact by name.
Allow updating the phone number or email of an existing contact.
Use methods like:
firstEntry(), lastEntry()
lowerEntry(), higherEntry()
descendingMap()

Exercício: Agenda Telefônica com TreeMap (NavigableMap)
Crie um programa que gerencie uma agenda telefônica usando TreeMap ou NavigableMap.

Requisitos:
Crie uma classe Contact com:
String name
String phoneNumber
String email

No programa principal:
Crie um NavigableMap<String, Contact>, onde a chave será o nome do contato e o valor será o próprio objeto Contact.

Adicione pelo menos 5 contatos.
Exiba todos os contatos em ordem alfabética por nome.
Mostre o primeiro e o último contato na agenda telefônica.
Dado um nome, mostre:
o contato imediatamente anterior (lowerEntry)
o contato imediatamente seguinte (higherEntry)
Exiba a agenda telefônica em ordem inversa (use descendingMap()).

Desafios Extras:
Permita a remoção de um contato pelo nome.

Permita a atualização do número de telefone ou email de um contato existente.
Use métodos como:
firstEntry(), lastEntry()
lowerEntry(), higherEntry()
descendingMap()

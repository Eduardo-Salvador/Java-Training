Exercise – Student Sorting:
Create a program to manage a list of students (Student) in a school.
Implement the system so that it is possible to sort the students in different ways.

Requirements:
Create the Student class with the following attributes:
String name
Integer age
Double grade (final grade)
Make the Student class implement Comparable<Student>,
so that the natural order of the objects is by name in alphabetical order.
Create two comparators (Comparator<Student>) to sort:
By ascending age
By descending grade
Create a main class StudentManager that:
Creates a list (List<Student>)
Adds 5 different students

Shows:
The original list
The list sorted by name (Comparable)
The list sorted by age (Comparator)
The list sorted by descending grade (Comparator)

Rules and Tips:
Use Collections.sort() for sorting with Comparable.
Use list.sort() for sorting with Comparator.
Use the forEach(System.out::println) method to print the list.
Keep the code well organized and modularized.

Exercício – Ordenação de Alunos:
Crie um programa para gerenciar uma lista de alunos (Student) em uma escola.

Implemente o sistema de forma que seja possível ordenar os alunos de diferentes maneiras.

Requisitos:
Crie a classe Student com os seguintes atributos:
String name
Integer age
Double grade (nota final)
Faça com que a classe Student implemente a interface Comparable<Student>,
para que a ordem natural dos objetos seja por nome em ordem alfabética.

Crie dois comparadores (Comparator<Student>) para ordenar:
Por idade crescente
Por nota decrescente
Crie uma classe principal StudentManager que:
Crie uma lista (List<Student>)
Adicione 5 alunos diferentes

Mostre:
A lista original
A lista ordenada por nome (Comparable)
A lista ordenada por idade (Comparator)
A lista ordenada por nota decrescente (Comparator)

Regras e Dicas:
Use Collections.sort() para ordenar com Comparable.
Use list.sort() para ordenar com Comparator.
Use o método forEach(System.out::println) para imprimir a lista.

Mantenha o código bem organizado e modularizado.

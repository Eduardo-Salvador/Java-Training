Exercise – City Management:
You must create a small system that manages cities in a country.
The goal is to practice using Iterators and converting between Lists and Arrays.

Requirements:
Create a City class with the following attributes:
Sequence name
Integer population
Create a CityManager class with the following methods:
addCity(List<City> cities, City city)
Adds a new city to the list.
Displays a message confirming the addition.
removeSmallCities(List<City> cities, int minPopulation)
Use an Iterator to remove all cities with a population less than minPopulation.
Displays the removed cities.
convertListToArray(List<City> cities)
Converts a list into an array (City[]).
Displays the contents of the array.
convertArrayToList(City[] cityArray)
Convert the array back into a list.
Display the contents of the new list.

Tips:
Use Iterator<City> iterator = cities.iterator(); to iterate and remove.
Remember to use iterator.remove() inside the loop, never cities.remove() directly.

To convert:
List → Array: City[] array = cities.toArray(new City[0]);
Array → List: List<City> list = new ArrayList<>(Arrays.asList(array));

Exercício – Gerenciamento de Cidades:
Você deve criar um pequeno sistema que gerencie cidades em um país.
O objetivo é praticar o uso de iteradores e a conversão entre listas e arrays.

Requisitos:
Crie uma classe City com os seguintes atributos:
Nome sequencial
População (Integer)
Crie uma classe CityManager com os seguintes métodos:
addCity(List<City> cities, City city)
Adiciona uma nova cidade à lista.
Exibe uma mensagem confirmando a adição.
removeSmallCities(List<City> cities, int minPopulation)
Usa um iterador para remover todas as cidades com população menor que minPopulation.
Exibe as cidades removidas.
convertListToArray(List<City> cities)
Converte uma lista em um array (City[]).
Exibe o conteúdo do array.
convertArrayToList(City[] cityArray)
Converte o array de volta para uma lista.
Exiba o conteúdo da nova lista.

Dicas:
Use Iterator<City> iterator = cities.iterator(); para iterar e remover elementos.
Lembre-se de usar iterator.remove() dentro do loop, nunca cities.remove() diretamente.

Para converter:
List → Array: City[] array = cities.toArray(new City[0]);
Array → List: List<City> list = new ArrayList<>(Arrays.asList(array));
